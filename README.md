# MindMap

MindMap is a single-file, browser-based visual thinking tool for creating, organizing, and presenting ideas as interactive maps.

## Description
MindMap helps you plan projects, study topics, and organize notes by building node-based diagrams in a canvas UI. It runs fully in the browser with no build step, and supports editing, layout switching, styling, search, and export workflows.

## Functionalities
- Create and edit maps with root, child, and sibling nodes.
- Move, resize, align, distribute, duplicate, copy/paste, and delete nodes.
- Use multiple layouts: Mind Map, Radial, Tree (Left-to-Right), Tree (Top-to-Bottom), Org Chart, and Circular.
- Add rich node content: labels, badges, links, images, footer notes, and checklists.
- Create cross-links between nodes for non-hierarchical relationships.
- Style nodes and edges (color, shape, font, edge style, edge width, and edge color).
- Search nodes and navigate matches quickly.
- Use multi-tab canvases in one session.
- Collapse/expand branches, including step-by-step expansion.
- Save/load maps in browser storage.
- Import/export JSON for sharing and backup.
- Export map as PNG image.
- Use undo/redo and common keyboard shortcuts for faster editing.

## Access Without Downloading
If GitHub Pages is enabled for this repository, users can open the app directly in the browser:
- Main app link: `https://jattakachora.github.io/MindMap/`
- Direct file link: `https://jattakachora.github.io/MindMap/mindmap.html`

If the link does not open yet, enable GitHub Pages in repository settings:
1. Go to `Settings` -> `Pages`.
2. Set source to `Deploy from a branch`.
3. Select `main` branch and `/ (root)` folder.

## Run Locally
1. Clone or download this repository.
2. Open `mindmap.html` in any modern browser.

No installation is required.
