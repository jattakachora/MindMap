# MindMap

Interactive browser-based mind mapping app.

## File
- `mindmap.html`: Main application file.

## How to use
1. Open `mindmap.html` in any modern web browser.
2. Create and connect nodes using the on-screen toolbar controls.
3. Save or export from the app UI when needed.

## Notes
- No build step or installation is required.
- Best experience is on a desktop browser.
